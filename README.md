# Deepfake Detection System

## 📌 Overview
This project detects whether an image/video is real or deepfake using a trained deep learning model.

## 🚀 Features
- Upload image/video for detection
- Deepfake classification with confidence score
- User-friendly frontend (React)
- Backend powered by Flask

## 🛠 Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Python (Flask)
- ML Model: XceptionNet
